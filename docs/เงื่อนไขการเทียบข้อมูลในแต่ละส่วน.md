เงื่อนไขการเทียบข้อมูลในแต่ละส่วน
1. ข้อมูลนำเข้า คือข้อมูลเดิม จากการ import ไฟล์ xlsx/csv ที่มีอยู่ 6 Sheet ในไฟล์เดียว ได้แก่ ข้อมูลพื้นฐาน ทุนมนุษย์ ทุนกายภาพ ทุนการเงิน ทุนธรรมชาติ และทุนทางสังคม
2. การสำรวจ คือ ข้อมูลการบันทึกข้อมูลจากการสำรวจใหม่ (ในการบันทึกไม่ต้องอ้างอิงจากการข้อมูลนำเข้า สามารถเพิ่มการสำรวจได้เลย) มีอยู่ 7 ส่วน ได้แก่ ข้อมูลพื้นฐาน ทุนมนุษย์ ทุนกายภาพ ทุนการเงิน ทุนธรรมชาติ ทุนทางสังคม และความพึงพอใจ ข้อเสนอแนะและความคิดเห็น

เมื่อเพิ่มการสำรวจแล้ว ต้องมี Logic ในการเทียบข้อมูลใหม่ กับข้อมูลเดิม โดยกำหนดการเทียบ จากข้อความถามในการสำรวจ กับหัวข้อคอลัมน์ ในไฟล์ import ดังนี้
1. ให้นำข้อมูลจากการสำรวจเป็นหลัก (โดยให้กำหนดค่าคะแนนแต่ละข้อขึ้นมาใหม่ได้เลย แต่ละทุน ต้องได้ครบ 100 คะแนน แล้วหาค่าเฉลี่ย ในแต่ละทุน และเอาทุน 5 ด้าน มาหาค่าเฉลี่ย เป็นรายคน และรายครัวเรือน) 
2. ในการเทียบเงื่อนไขในแต่ละข้อ หากในการสำรวจมีข้อที่ไม่ตรงกับไฟล์นำเข้า ให้คิดคะแนนข้อนี้ด้วย (ไม่ต้องเทียบกับข้อมูลเดิมเพื่อหาเกณฑ์การขยับฐานะขึ้นของครัวเรือน) หากมีข้อที่ตรงกัน ให้เอาการสำรวจใหม่ ไปเทียบกับข้อมูลนำเข้าว่าดีขึ้น คงที่ หรือแย่ลง
3. แต่ละ Sheet มี รหัสบ้าน บ้านเลขที่ หมู่ที่ ตำบล อำเภอ จังหวัด กำกับอยู่ หากจะสรุปค่าคะแนน ของแต่ละครัวเรือน แต่ละทุน เพื่อมาเทียบกับ Sheet ข้อมูลพื้นฐาน ก็สามารถทำได้เลย

การเทียบข้อมูลจากการสำรวจ กับข้อมูลนำเข้า แบบตอนที่ ข้อ ในการสำรวจ กับ Sheet คอลัมน์ และข้อมูลใน rows ดังนี้

	2. ชื่อโมเดล /ไม่ต้องเทียบ ให้เพิ่มรายการใหม่ได้เลย
	3. พื้นที่ ตำบล อำเภอ จังหวัด /ไม่ต้องเทียบ ให้เพิ่มรายการใหม่ได้เลยให้เป็นข้อมูลที่เชื่อมกับโมเดล ว่าโมเดลนี้อยู่ในพื้นที่ไหน

	8. จังหวัด เทียบกับ คอลัมน์ H (จังหวัด)
